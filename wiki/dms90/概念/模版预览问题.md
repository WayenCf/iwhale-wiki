# 模版预览问题

> DMS 模版在设计器中预览与实际生成效果不一致的问题。

## 在现场问题中的角色

### 文字截断

- [[wiki/dms90/来源/2024-08-14 MPT模版设计器预览与接口生成不一致]]：JasperReports 后台计算文字宽度与 PDF 实际宽度不一致导致截断
- 解决：`net.sf.jasperreports.export.pdf.force.linebreak.policy=true`

### 预览与 Test 不一致

- [[wiki/dms90/来源/2025-03-11 模版预览和模版Test不一致]]：预览获取最新模版，Test 使用历史模版
- 解决：新增历史模块开关判断

### 字体导致数据不展示

- [[wiki/dms90/来源/2024-04-10 缅甸MPT模版预览不显示数据]]：字体版本升级后模版数据不展示

## 常见故障模式

1. **文字宽度计算偏差** → PDF 文本截断 → 配置 force.linebreak.policy
2. **历史模版问题** → Test 使用旧模版 → 增加开关判断
3. **字体兼容** → 数据不展示 → 回退字体版本

## 关键要点

- 预览接口 `release/getSnapshot` 获取最新模版
- Test 接口走 make 流程可能使用历史模版
- 设计器预览与 PDF 生成使用不同的渲染引擎
