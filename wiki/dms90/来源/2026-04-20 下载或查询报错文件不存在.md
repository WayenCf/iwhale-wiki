# 下载或查询报错文件不存在

> 来源：微服务平台团队-陈飞个人文档

---

## 问题一：下载/查询报错文件不存在

### 问题现象

下载或查询文件时报错，提示文件不存在。

### 问题原因

[[wiki/dms90/实体/MongoDB]] 的 `TaskRecord` 集合中存在 `_id` 重复的数据记录，导致查询/下载时定位异常。

### 解决方案

删除重复的 TaskRecord 记录：

```javascript
// 先查询重复记录
db.getCollection("TaskRecord").find({
  "_id": { $regex: /m-duka.jpg/ }
}).sort({ "updateDate": -1 });

// 删除重复记录
db.getCollection("TaskRecord").deleteOne({
  "_id": "D:\\dmsFile\\default\\crm\\crm-duka.jpg"
});

// 返回 1 代表成功
{ "acknowledged": true, "deletedCount": 1 }
```

## 问题二：扫描 XML 文件报错

### 问题现象

扫描 XML 文件报错：`failed to parse the xml file`

### 问题原因

XML 文件格式有问题，文件中存在未转义的特殊字符。

### 解决方案

检查 XML 文件中是否存在转义字符或特殊字符，进行修正后重新上传。
