# 代码分析：dms-web

> DMS Web 层精读，提供 REST API、安全配置、请求拦截、全局异常处理、优雅停机等能力，是整个 DMS 系统对外的入口模块。

## 模块职责

- **根包**：`com.iwhalecloud.ftf.dms.web`
- **运行模式**：`independent`（独立部署）/ `integrated`（嵌入 Portal）
- **核心类数量**：约 55 个主源文件

### 包结构

| 子包 | 类数 | 职责 |
|------|------|------|
| `controller` | 12 | 核心业务 REST 控制器 |
| `controller.api` | 7 | 对外 API 控制器（OpenAPI v3 注解） |
| `config` | 12 | [[wiki/dms90/实体/Spring Boot|Spring Boot]] Security 配置、Properties |
| `filter` | 6 | 拦截器/过滤器 |
| `exception` | 6 | 自定义异常 + 全局异常处理器 |
| `jasperserver` | 7 | [[wiki/dms90/实体/JasperReports|JasperServer]] 兼容层 |
| `automation` | 1 | 定时任务（存储统计） |

## 核心类

### Controller 层

- **ResourceController**（`/resources`）：CRUD 资源，调用 ResourceService，安全校验 SecureUtil.checkParams()
- **PublishController**（`/publish`）：发布资源，含 `defend()` 防 HTML/SQL 注入
- **DelegateController**（`/delegate`）：Token 下载，[[wiki/dms90/概念/多租户隔离|多租户]] 切换
- **TenantController**（`/tenants`）：租户 CRUD，手动角色校验
- **CacheRefreshController**（`/cache`）：刷新 [[wiki/dms90/概念/缓存架构|本地缓存]]
- **MetricController**（`/metric`）：加载/清除 [[wiki/dms90/实体/Sentinel|Sentinel]] 降级规则
- **SearchController**（`/search`）：全文搜索，[[wiki/dms90/实体/JasperReports|JasperReports]] 关联

### 拦截器

- **FlowLimitationInterceptor**：文档生成路径流控，超 warningThreshold 返回 413，超 awaitThreshold sleep 3s
- **FileTypeInterceptor**：上传文件后缀白名单校验
- **RequestCounterInterceptor**：请求计数，配合 GracefulShutdown
- **ParameterMapResetInterceptor**：ThreadLocal 清理防泄漏
- **UploadCapacityFilter**：上传容量预检，CAS 无锁管理
- **AdminCheckFilter**：多租户下禁止 admin 账号
- **DMSCsrfFilter**：CSRF 防护（integrated 模式）

## 关键发现

1. **双模式安全配置**：SecurityConfiguration（independent）禁用 CSRF + Form Login；PortalDependedSecurityConfiguration（integrated）HTTP Basic + 自定义 CSRF
2. **[[wiki/dms90/概念/异常体系|全局异常处理]]**：AppExceptionHandler 映射 11 种异常到 HTTP 状态码，统一 ErrorMessage 格式
3. **上传容量控制**：UploadCapacityManager 基于 AtomicLong CAS 无锁管理 + [[wiki/dms90/实体/Sentinel|Sentinel]] 线程级流控
4. **优雅停机**：GracefulShutdown 监听 ContextClosedEvent，内置 Tomcat pause 30s，外置 Tomcat 等待 RequestCounter=0
5. **JasperServer 兼容**：REST/SOAP 代理层，XML 数据格式（JAXB），硬编码版本 6.2.0

## 设计模式

- **AOP 访问日志**：LogAroundMethod 拦截 controller.api 包，记录方法名/用户/耗时
- **条件化装配**：`@ConditionalOnProperty`/`@ConditionalOnBean` 大量使用
- **无锁并发**：UploadCapacityManager CAS 循环 + 自适应退避
- **异常优化**：4 个高频异常覆写 fillInStackTrace() 跳过堆栈填充
