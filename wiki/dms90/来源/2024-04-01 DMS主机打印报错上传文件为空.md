# DMS主机打印报错上传文件为空

## 问题现象

业务主机调用 [[wiki/dms90/来源/2026-04-19 DMS项目概览|DMS]] 失败，[[wiki/dms90/来源/2026-04-19 DMS项目概览|DMS]] 主机打印报错 `The uploading file [file] is empty`。

## 问题分析

使用 arthas 跟踪现场，发现上传的文件内容为空。

## 问题验证

1. 本地复现问题：当上传的文件内容为空时，会提示 `The uploading file [file] is empty`
2. 功能正常，需要业务侧上传的文件里面有内容才行

## 关键要点

- [[wiki/dms90/来源/2026-04-19 DMS项目概览|DMS]] 对上传空文件有校验
- 此报错为正常的业务校验，非系统 bug
- 需业务侧确保上传文件有内容
