# DMS 项目概览

> DMS (Document Management System) — 中讯鲸云 iWhaleCloud 的企业级 Java/Spring Boot 文档管理平台。版本 9.2.21-SNAPSHOT。

| 父 POM | ztesoft-paas-parent 9.4-SNAPSHOT |

## 关键要点

### 架构

- **Maven 多模块**，16 个子模块：dms-common / dms-service-api / dms-service-impl / dms-repo-api / dms-repo-mongodb / dms-repo-jdbc / dms-repo-spool / dms-web / dms-restful-api / dms-docmaker / dms-jasperreport-ext / dms-boot / dms-customization / dms-embed / dms-fonts / dms-designer-plugins / dms-demo
- **分层架构**：Controllers (dms-web) → Services (dms-service-api/impl) → Repositories (dms-repo-api/*)
- **[[wiki/dms90/概念/双存储适配|双存储适配]]**：通过 Spring Profiles 切换 — `mongodb`（默认）、`spool`（文件系统）、`portal`（认证集成）
- **入口**：`DmsApplication.java`，`@SpringBootApplication(scanBasePackages = "com.iwhalecloud.ftf.dms")`

### 核心控制器

| 控制器 | 职责 |
|--------|------|
| `DocumentController` | 核心：文档 CRUD、批量生成、下载（最大的控制器） |
| `FileController` | 文件上传/下载 + 限流 |
| `SearchController` | 全文搜索（Elasticsearch） |
| `DelegateController` | 委托操作 |
| `MaterialController` | 素材管理 |
| `FolderController` | 文件夹管理 |
| `PublishController` | 发布管理 |

### 限流与流控

- 使用 **Alibaba Sentinel 1.7.2**
- 上传/下载并发线程限制、总字节限制
- 配置在 `dmsConfig.properties`

### 关键依赖

| 依赖 | 版本 | 用途 |
|------|------|------|
| Spring Boot | 2.x | 核心框架 |
| JasperReports | 6.10.0 | 文档/账单生成 |
| Apache PDFBox | 2.0.28 | PDF 处理 |
| MongoDB driver | 4.4.2 | 文档存储 |
| Alibaba Sentinel | 1.7.2 | 流控/限流 |
| ZXing | 3.4.0 | 二维码/条码生成 |

### 产品标准配置

- [[wiki/dms90/来源/2024-06-04 产品标准配置添加]]：新增 DMS 产品标准配置项

### 前端

- 位于 `dms/dms-web/src/main/webapp/`
- 使用 **Fish** UI 框架（自研企业框架）
- 构建工具：Grunt + Gulp

### 部署

- Dockerfile：OracleLinux 9 + Java 21
- JVM：60% RAM 分配，G1GC，OOM 时 heap dump
- 默认端口：8082，上下文路径：`/dms`
