# 产品标准配置添加

## 需求背景

新增 [[wiki/dms90/来源/2026-04-19 DMS项目概览|DMS]] 产品标准配置项。

## 新增配置项

### 模版并发处理线程数配置

```properties
# 模版并发处理线程数
ftf.dms.batch.executor-pool-size=
```

默认值根据 CPU 核数动态计算，一般不配置。

### 模版处理超时时间配置

```properties
# 模版处理超时时间
ftf.dms.batch.executor-time-out=
```

单位为秒，默认 0 表示不超时。

### 批量结果保存数据库开关

```properties
# 批量结果是否保存到数据库
ftf.dms.batch.result.saveToDB=
```

默认 true，保存到数据库；设为 false 则不保存。

### 文档存储加密

```properties
# 文档存储是否加密
ftf.dms.spool.fileId.encrypt=
```

默认 true，上传文件返回加密文件 ID。

## 关键要点

- 线程池大小可根据 CPU 核数动态调整
- 超时时间设为 0 表示不限制
- 批量结果可选择不保存到数据库以减少 IO
- 文件 ID 加密是安全合规要求
