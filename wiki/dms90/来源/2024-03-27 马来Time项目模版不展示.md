# 马来Time项目模版不展示

## 问题现象

打开 [[wiki/dms90/来源/2026-04-19 DMS项目概览|DMS]] 模版菜单页面，模版不展示。

## 问题原因

对比本地数据库模版表 `DMS_RESOURCE` 表缺少基础数据导致。缺少以下目录记录：
- `/Template`
- `/Sub-Template`
- `/DataAdapter`
- `/Resource`
- `/Picture`
- `/Multimedia`
- `/Others`

## 解决方案

新增缺失的模版记录，通过 SQL 插入 `DMS_RESOURCE` 表的基础数据：

```sql
INSERT INTO DMS_RESOURCE
(ID, UPDATE_DATE, CREATION_DATE, USER_ID, VERSION, RESOURCE_TYPE, PERMISSION_MASK, DESCRIPTION, LABEL, FILE_ID, DEPENDENCIES, RELEASE_VERSION, CODE)
VALUES('/Template', TIMESTAMP '2024-01-06 17:05:31.000000', TIMESTAMP '2024-01-06 17:05:31.000000', '0', 0, 'folder', 1, 'System Default Folder.', 'Template', NULL, NULL, NULL, NULL);
-- 其余目录记录类似...
```

## 关键要点

- [[wiki/dms90/来源/2026-04-19 DMS项目概览|DMS]] 模版展示依赖 `DMS_RESOURCE` 表的基础目录数据
- 新环境搭建时需确保基础数据完整
- `RESOURCE_TYPE='folder'` 且 `USER_ID='0'` 为系统默认目录记录
