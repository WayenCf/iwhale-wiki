# 阿富汗下载PDF报错超出限制条数

## 问题现象

下载 PDF 报错：

```
com.iwhalecloud.ftf.dms.repo.mongodb.exception.MongoException: The count of the results returned from database exceeds the limit [10,000]
```

## 问题原因

[[wiki/dms90/实体/MongoDB|MongoDB]] 数据过多，一次查询的数量超出了 `ftf.dms.mongo.result-limit` 的设定值。

## 解决方案

修改查询条数配置值：

```properties
ftf.dms.mongo.result-limit=2000
```

目前解决方案是在不影响性能的情况下暂时调大，后期建议定时清理。

## 关键要点

- `ftf.dms.mongo.result-limit` 控制 [[wiki/dms90/实体/MongoDB|MongoDB]] 单次查询返回的最大条数
- 数据量增长后需定期清理历史数据
- 调大限制值可能影响性能
