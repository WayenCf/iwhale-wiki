# XML特殊字符导致PDF生成失败

## 问题现象

业务生成账单后调用 [[wiki/dms90/来源/2026-04-19 DMS项目概览|DMS]] 打印 PDF 失败，日志报错：

```
Fail to parse the xml file
```

## 问题原因

XML 中含有特殊字符，如 `&`、`<`、`>` 等，导致 XML 解析失败。

## 解决方案

1. 在生成 XML 时对特殊字符进行转义处理
2. XML 特殊字符转义对照：
   - `&` → `&amp;`
   - `<` → `&lt;`
   - `>` → `&gt;`
   - `"` → `&quot;`
   - `'` → `&apos;`

## 关键要点

- XML 规范定义了 5 个预定义实体引用
- 特殊字符未转义是 PDF 生成失败的常见原因
- 需在业务侧生成 XML 时统一处理特殊字符
