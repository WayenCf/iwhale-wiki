# iWhaleCloud

> 中讯鲸云（鲸云），ZTE 旗下云计算/软件服务公司，DMS 文档管理系统的开发商。

## 基本信息

| 字段 | 值 |
|------|-----|
| 全称 | iWhaleCloud（中讯鲸云） |
| 母公司 | ZTE（中兴通讯） |
| PaaS 平台 | ZTEsoft PaaS（parent version 9.4-SNAPSHOT） |
| 代码包基路径 | `com.iwhalecloud.ftf.dms` |

## 产品体系

- **DMS** (Document Management System) — 企业级文档管理平台，当前版本 9.2.21-SNAPSHOT
- **Fish UI** — 自研企业前端框架，用于 DMS Web 界面

## 技术栈特征

- Java 8 源码 + Java 21 生产运行时
- Spring Boot 2.x + ZTEsoft PaaS Parent
- 多后端存储策略（MongoDB / Spool / JDBC）
- Docker 部署（OracleLinux 9 基础镜像）
