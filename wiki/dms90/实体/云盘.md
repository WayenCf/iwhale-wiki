# 云盘

> DMS 的文件存储后端，通过挂载方式接入，用于存储上传的文件和生成的账单 PDF。

## 在现场问题中的角色

### 挂载配置丢失

- [[wiki/dms90/来源/2024-03-21 TM项目DMS云盘维护]]：DMZ 升级后云盘挂载消失，需重新配置

### 用户权限不足

- [[wiki/dms90/来源/2024-03-12 卢旺达文件上传下载失败]]：`zpaas` 用户无云盘读写权限
- [[wiki/dms90/来源/2024-04-17 卢旺达第二套DMS应用部署问题]]：DMS2 容器用户 `zpaas` 权限不足
- [[wiki/dms90/来源/2024-07-27 TM云盘问题]]：云盘权限问题（错误码 7062411）

### IO 性能瓶颈

- [[wiki/dms90/来源/2024-04-02 格鲁生产环境DMS出账较慢]]：`uploadFile` 调用云盘代码较慢

### 路径配置问题

- [[wiki/dms90/来源/2026-04-20 离线上传文件不扫描]]：`CLOUD_NODE_ID` 环境变量缺失导致云盘路径异常（`file//` 双斜杠）

### 容量不足

- 文件上传失败时需检查云盘容量

## 常见故障模式

1. **挂载丢失** → DMZ 升级/重启后需重新配置
2. **权限不足** → `zpaas` 用户无读写权限 → 修改为 `root` 或开放权限
3. **容量满** → 文件上传失败 → 扩容
4. **IO 慢** → 出账性能下降 → 新增实例

## 故障排查步骤

1. 检查 DMS 应用用户是否为 `zpaas`
2. 检查 `zpaas` 用户是否有读写权限
3. 检查云盘容量剩余大小
4. 检查挂载路径是否正确
