# Raw 原始资料目录

> **只读目录**：此目录存放原始资料文件，由用户维护，Claude 只读。

---

## 目录结构

```
raw/
├── README.md           ← 本文件：raw 目录说明
├── 素材/               ← 图片、附件等媒体资源
└── {项目名}/           ← 按项目分类的原始资料
    └── {项目名}/
```

## 用途

1. **存放原始资料**：现场问题文档、代码分析、设计文档等
2. **溯源引用**：wiki 中的来源页通过 `[[raw/...]]` 链接到原始文件
3. **知识来源**：Claude 摄取新资料时从此目录读取

## 使用方式

### 添加新资料

1. 将原始文件复制到 `raw/{项目名}/` 目录
2. 告诉 Claude："摄取 raw/{项目名}/xxx.md"
3. Claude 会自动：
   - 读取文件内容
   - 创建 `wiki/{项目}/来源/YYYY-MM-DD 标题.md` 摘要页
   - 在来源页顶部添加 `[[raw/{项目}/xxx.md]]` 溯源链接
   - 更新相关实体/概念页
   - 更新 index.md 和 log.md

### 文件命名建议

- **现场问题**：`YYYYMMDD-{项目}-{问题简述}.md`
- **代码分析**：`代码分析-{模块名}.md`
- **设计文档**：`{模块名}-{文档类型}.md`

## 注意事项

- ⚠️ **不要手动修改** `raw/` 中的文件后，指望 Claude 自动更新 wiki
- ⚠️ **不要删除** `raw/` 中已被 wiki 引用的文件（会导致断链）
- ✅ **如需更新**：重新执行摄取操作，Claude 会更新 wiki 内容
